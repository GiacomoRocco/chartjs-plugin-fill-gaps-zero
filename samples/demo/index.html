chartjs-plugin-fill-gaps-zero

Chart.js plugin - This plugin fill gaps with zero in X time axis for line chart.

Plugin Enable Example

                {
                    x: {
                            fillGapsWithZero: true,
                            minUnit: 'day',
                            type: 'time',
                            time: {
                                unit: 'day'
                            },
                            grid: {
                                display: false
                        }
                }

Plugin Not Enable Example

                    {
                        x: {
                                fillGapsWithZero: false,
                                minUnit: 'day',
                                type: 'time',
                                time: {
                                    unit: 'day'
                                },
                                grid: {
                                    display: false
                            }
                    }
